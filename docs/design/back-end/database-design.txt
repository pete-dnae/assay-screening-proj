-------------------------------------------------------------------------------
SUMMARY
-------------------------------------------------------------------------------

Concentration           Stock and final dilution ratio, plus units.
ConcreateReagent        E.g. BSA.
BufferMix               Table containing concrete reagents and volumes.
MixedReagent            Meta-reagent comprising a BufferMix + concentration.
PlaceHolderReagent      Generic type of reagent - e.g. Primer or Template.
MasterMix               Table containing all types of reagents and volumes. 

Gene                    E.g. uidA
Primer                  Either a forward or reverse primer. E.g. Eco66
PrimerPair              A matched pair of fwd/rev primers.
PrimerKit               Collection of all primer pairs used in an experiment.

Organism                E.g. Enterococcus faecalis.
Arg                     E.g. vanB.
Strain                  E.g. ATCC BAA-1705.
StrainKit               Collection of all strains used in an experiment.

CyclingPattern          Times, temparatures and counts.

RowPattern              Describes a repeat pattern for a given row on a plate.
AllocationInstructions  Repeat and pattern recipe for a plate.
Plate                   Everything pertaining to a single plate.
Experiment              Everything pertaining to one experiment.
